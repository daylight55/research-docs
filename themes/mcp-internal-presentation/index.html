MCP internal presentation

MCPを開発現場でどう使うべきかを説明するための調査テーマ。

このテーマでは、スライド、詳細調査メモ、source links、research task checklist を同じ親テーマの下にまとめる。成果物が増えた場合も、テーマ配下の artifact として追加する。

Theme objective

  • MCPの概要、動作フロー、歴史、概念、認証仕様、接続プロトコルを説明できる形にする。
  • Agent Skills と MCP の関係を、手順と操作面の分離として整理する。
  • 公式仕様、一次情報、主要リポジトリ、実装メモを発表用スライドへ圧縮する。

Child artifacts

  • Slide deck: 発表用Marpスライド
  • Research notes: 調査メモと論点整理
  • Source links: 公式仕様、一次情報、関連リポジトリ
  • Research task checklist: 未解決項目と検証済み項目
